# 商业化布局重构设计文档

## 1. 背景

yuleCommunity 平台现有导航栏 12 项平铺，移动端体验碎片化，缺少商业化引导。重构目标是提升品牌专业感、优化移动端体验、增加商业转化入口。

---

### 2.3 响应式断点设计

| 断点 | 宽度 | 导航模式 | 布局特征 |
|------|------|---------|---------|
| `xs` | < 480px | 底部 Tab + 汉堡菜单 | 单列，紧凑间距 |
| `sm` | 480-768px | 底部 Tab + 汉堡菜单 | 单列，标准间距 |
| `md` | 768-1024px | 顶部缩略导航 | 两列，侧栏可折叠 |
| `lg` | 1024-1280px | 顶部完整导航 | 两列/三列 |
| `xl` | > 1280px | 顶部完整导航 | 多列，宽屏优化 |

## 3. 组件设计

### 3.1 Navbar 重构

**组件：** `MobileNav.tsx`（新增），`Navbar.tsx`（重构）

**桌面端 Navbar 变化：**
- 12 项 → 5 项主要 + 1 个「更多」下拉
- 添加 `MoreHorizontal` 下拉菜单组件
- 添加「免费加入」CTA 按钮
- 已登录显示用户头像

**移动端新增 BottomTabBar：**
- 固定底部的 `position: fixed; bottom: 0`
- 5 个 Tab，每个带图标+文字
- 使用 lucide-react 图标
- 当前页面高亮
- 底部留安全区适配 iPhone 刘海

**「更多」下拉组件 `MoreDropdown.tsx`：**
- 分组显示子菜单（带图标）
- 点击后跳转+自动关闭
- 点击外部关闭
- 键盘 Escape 关闭

**汉堡菜单（移动端）`MobileDrawer.tsx`：**
- 从右侧滑入的面板
- 显示全部 12 个导航项
- 顶部带搜索框
- 底部带用户信息和设置

### 3.2 DevHub 统一布局

**组件：** `DevHubLayout.tsx`（新增）

所有 DevHub 子页面共用布局：

```
┌──────────────────────────────────────────┐
│ ← 返回  |  📖 AutoSAR 开发者中心  |  子页面标题 │  ← 内页 Header
├──────────────────────────────────────────┤
│                                          │
│          子页面内容（slot）                │
│                                          │
└──────────────────────────────────────────┘
```

- 仅在 DevHub 子页面使用
- 统一间距 padding: `px-4 md:px-8`
- 统一 max-width: `max-w-7xl`
- 面包屑导航一致性

### 3.3 Spec 浏览器响应式优化

**当前问题：** 移动端两个 `<select>` 下拉选择模块和 API，体验割裂。

**优化方案：** 底部 Sheet 面板（滑动抽屉）

```
┌──────────────────────────────┐
│                              │
│        API 详情内容           │
│                              │
│                              │
├──────────────────────────────┤
│  📁 选择模块/API  ▲          │  ← 底部把手
└──────────────────────────────┘

点击后上滑展开：
┌──────────────────────────────┐
│  🔍 搜索 API...              │
├──────────────────────────────┤
│  ▼ MCAL                     │
│    ├ Can                    │
│    │ ├ Can_Init  ⭐          │
│    │ ├ Can_Write            │
│    │ └ ...                  │
│    ├ Dio                    │
│    └ ...                    │
├──────────────────────────────┤
│  ▼ ECUAL                    │
└──────────────────────────────┘
```

- 底部 Sheet 使用 framer-motion 动画
- 拖拽关闭（模仿 iOS 交互）
- 保留当前模块/API 高亮
